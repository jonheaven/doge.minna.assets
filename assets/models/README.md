# 3D Models

This folder contains all 3D model assets for the doge.みんな project.

## Model Files

- `old_tree.glb` - Ancient tree model (gelmi.com.br)
- `bonsai.glb` - Mystical bonsai tree
- `animated_dog_shiba_inu.glb` - Primary Kabosu (quander)
- `shiba_inu_doggy.glb` - Alternate Kabosu (aaadragon)
- `moon.glb` - Moon model (optional)

## Adding New Models

1. Place GLB files in this directory
2. Update `js/config.js` with new model paths
3. Add loading logic in appropriate manager classes
4. Update attributions in `js/config.js` and main README
